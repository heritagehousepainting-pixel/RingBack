# FirstBack — Focus (north star)

**Until the first $99 invoice, the only metrics that matter:**
1. **Forwarding-confirmation rate** — do contractors successfully route missed calls to FirstBack?
2. **Calls-answered-by-AI rate** — is the AI actually picking up and handling those calls?

Everything else is downstream of these two. If a work session opens with refactoring a CRM
integration, the Telnyx provider, Outlook mail, or an exotic growth play type — redirect. That
code exists and is gated; the scarce resource is attention, not lines of code.

**The two real prerequisites to revenue (neither is code):**
- **Deploy voice** — set `VOICE_PUBLIC_URL`, stand up the `voice_service.py` sidecar, pick a real
  `CONVERSATIONRELAY_VOICE`, make one test call. Nothing in the product works without it.
- **Cold outreach** — 20 contractors in one trade + one metro (painters or HVAC), via Facebook
  groups / Nextdoor Pros / supplier reps. Contractor #1 is the only milestone that validates anything.
